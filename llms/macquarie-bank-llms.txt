# Macquarie Bank

> Macquarie Bank Limited is an Australian authorised deposit-taking institution (ADI) and the retail/business banking arm of Macquarie Group (ASX:MQG). As a designated Consumer Data Right (CDR / Open Banking) data holder it exposes a public, unauthenticated Product Reference Data (PRD) API and CDS discovery endpoints, plus consent-based CDR banking data sharing and the DEFT payments platform through a registered developer portal. Generated by the API Evangelist enrichment pipeline (2026-07-20).

## APIs

- [Macquarie Bank CDR Product Reference Data API](https://www.macquarie.com.au/digital-banking/open-banking.html): Public, unauthenticated CDS Product Reference Data. Base https://api.macquariebank.io/cds-au/v1/banking. GET /banking/products and /banking/products/{productId}. Requires the x-v version header (confirmed live: HTTP 200, x-v:3, 21 products).
- [Macquarie Bank CDR Discovery Status API](https://www.macquarie.com.au/digital-banking/open-banking.html): Public CDS discovery. GET /discovery/status (live 200, x-v:1) and /discovery/outages.
- [Macquarie Bank CDR Banking Data Sharing API](https://www.macquarie.com.au/digital-banking/open-banking.html): Consent-based accounts/balances/transactions/payees/direct-debits/scheduled-payments. Requires CDR ADR accreditation + FAPI 1.0 Advanced OAuth2/OIDC over MTLS. Not openly callable.
- [Macquarie DEFT Payments API](https://developer.macquariebank.io/devportal/): Tokenised card, bank account and direct-debit payment APIs via the registered developer portal (Macquarie Connect). Card payments cease 1 October 2026.
